读取方法:(在访问CMOS寄存器时必须禁止所有的中断)

CMOS内存的地址口: 0x70
		  数据口: 0x71
现将要读/写的地址写到地址口，在将要读/写入的数据送入0x71.

读CMOS
#define CMOS_READ(addr)   ({outb(0x80|addr,0x70);inb(0x71);})  
BCD转2进制
#define BCD_TO_BIN(val)   ((val)=((val)&15)+((val)>>4)*10)
